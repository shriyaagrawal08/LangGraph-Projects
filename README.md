# 🚀 LangGraph Projects

A collection of practical projects exploring **LangGraph workflows, LLM-based systems, and prompt engineering techniques**.
This repo demonstrates how to build structured, stateful AI pipelines using modern LLM tooling.

---

## 📌 Projects Included

### 1. 🧮 BMI Calculator (Without LLM)

* Basic logic-driven application (no LLM dependency)
* Demonstrates structured flow and input handling

---

### 2. ⚙️ Fault Tolerance using Persistence

* Implements **state persistence in LangGraph**
* Ensures recovery and continuity in workflows
* Useful for production-grade AI pipelines

---

### 3. 😂 Joke Generator with Explanation

* Generates jokes using LLM
* Adds explanation layer → improves interpretability

---

### 4. ✍️ Blog Generation using Prompt Chaining

* Step-by-step blog creation pipeline
* Uses chained prompts for structured output
* Shows how to control LLM responses

---

### 5. 📝 UPSC Essay Evaluation using LLM

* Evaluates essays using LLM-based scoring
* Simulates real-world AI evaluation systems
* Focus on reasoning + structured feedback

---

### 6. 🔄 Review Handling System

* Processes and analyzes reviews
* Demonstrates workflow-based LLM usage

---

## 🛠️ Tech Stack

* **Python**
* **LangGraph**
* **LLMs (OpenAI / similar)**
* Prompt Engineering

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/LangGraph-Projects.git
cd LangGraph-Projects
```

### 2. Create virtual environment (recommended)

```bash
python -m venv venv
source venv/bin/activate   # Mac
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
